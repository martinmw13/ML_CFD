# ML_CFD
This repository contains the code developed for my thesis for the Nuclear Engineering degree. PCA and non-linear dimensionality reduction techniques are applied for Computational Fluid Dynamics (CFD) simulations. Please note that this work is currently in progress.

The code provided here implements various techniques for dimensionality reduction, focusing primarily on PCA (Principal Component Analysis) and exploring non-linear alternatives, such as, LLE and Isomap.

In the near future, I plan to extend this repository by incorporating an autoencoder for dimensionality reduction and potentially training a Convolutional Neural Network (CNN) for estimating the Reynolds number (Re) from CFD simulation snapshots.

The CFD data is not uploaded for size matters, if needed, contact me at martinmadridwagner@gmail.com.
